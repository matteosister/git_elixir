add(what \\ ".")
(function) # ↑

Adds elements to the index. The argument gets passed to the git add “…” command, so you can use wildcartd etc.

Example iex> GitElixir.Command.add ["add", "."] iex> GitElixir.Command.add(“src/“) ["add", "src/"]
